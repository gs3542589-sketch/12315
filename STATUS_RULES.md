# STATUS_RULES.md - 操作前汇报规则

## 汇报原则
**超过10秒的操作必须提前告知用户。**

## 需要汇报的场景
| 操作 | 汇报内容 |
|------|----------|
| 生成图片/视频 | "生成图片中，预计30秒" |
| 搜索信息 | "正在搜索，稍等" |
| 安装skill | "正在安装skill，等1分钟" |
| 执行长时间命令 | "执行中，预计X秒" |
| 批量操作 | "处理X个文件，需要Y分钟" |

## 汇报格式
```
[操作描述]，预计[时间]，稍等
```

## 不需要汇报的场景
- 读文件、查资料
- 写日志
- 短命令（<5秒）
- 已有明确进度的操作

## 进度更新
- 长时间操作每30秒更新一次进度
- 完成立即汇报结果
- 失败立即说明原因和下一步建议
