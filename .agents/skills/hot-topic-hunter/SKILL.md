# Hot Topic Hunter - 热点选题猎手

## 技能说明
聚合多平台热点数据，自动生成选题建议和爆款标题。

## 触发词
热点选题、今日热点、爆款选题、选题推荐、热点追踪

## 数据源（按优先级）

| 平台 | 数据源 | 更新频率 | 适用场景 |
|------|--------|----------|----------|
| 抖音 | douyin-hot / 第三方API | 实时 | 短视频选题 |
| 小红书 | xiaohongshu技能 | 实时 | 种草/生活方式 |
| B站 | bilibili-hot-monitor | 每小时 | 知识/深度内容 |
| 微博 | 热搜API | 实时 | 时事/热点评论 |
| 知乎 | 热榜API | 每小时 | 深度观点 |

## 工作流程

### Step 1: 收集赛道
询问用户：
```
请选择你的创作赛道：
1. 知识科普
2. 情感故事
3. 产品种草
4. 剧情向
5. 搞笑段子
6. 职场成长
7. 美食探店
8. 旅游vlog
9. 其他（请描述）
```

### Step 2: 抓取热点
根据赛道调用对应数据源：
- 调用 `xiaohongshu` 技能获取小红书热点
- 调用 `bilibili-hot-monitor` 获取B站热榜
- 其他平台用 web_search 补充

### Step 3: 分析筛选
**筛选标准**：
1. 与赛道相关性 ≥ 60%
2. 热度指数 > 阈值
3. 可创作性评估（是否有素材/可落地）

**输出格式**：
```
【热点选题报告】
生成时间：YYYY-MM-DD HH:mm

🔥 TOP 3 热点选题

1️⃣ [选题标题]
   - 热度：★★★★★
   - 来源：抖音热榜 #12
   - 相关度：85%
   - 建议角度：[具体切入点]
   - 推荐理由：[为什么适合你]

2️⃣ [选题标题]
   ...

3️⃣ [选题标题]
   ...

📊 数据来源
- 小红书：X条相关笔记
- B站：X个相关视频
- 微博：X条热搜

⚠️ 避坑提示
- [敏感词/风险点]
```

### Step 4: 生成标题
基于选题，用爆款公式生成5个标题：

**公式库**：
| 公式 | 模板 | 示例 |
|------|------|------|
| 二极管 | XX都可怕 | 创业的人都可怕 |
| 共情 | 救命/真心建议 | 救命！这个方法太绝了 |
| 数字 | 我XX岁/XX天 | 我30岁，存款为0 |
| 反差 | 说出来你可能不信 | 说出来你可能不信，我靠这个月入5万 |
| 悬念 | 逼自己看完 | 逼自己看完，改变你的一生 |
| 痛点 | 很变态但 | 很变态但很有效 |

**输出**：
```
【爆款标题候选】
1. [标题1] （公式：二极管）
2. [标题2] （公式：共情）
3. [标题3] （公式：数字）
4. [标题4] （公式：反差）
5. [标题5] （公式：悬念）

推荐首选：[标题X]
理由：[为什么这个最容易爆]
```

## 敏感词过滤

**自动规避**：
- 政治敏感词
- 平台违禁词
- 争议性话题

**替代方案**：
- 自动替换敏感词
- 提供安全角度建议

## 参数配置

### 热度阈值
- 高热度：> 100万播放/点赞
- 中热度：10-100万
- 潜力股：1-10万（竞争小）

### 时效性
- 实时热点：< 24小时
- 短期热点：1-3天
- 长期话题：> 1周

## 与其他技能联动

- 选定选题后 → 自动调用 `aigc-video-pipeline` 进入脚本创作
- 可配合 `viral-headline-writer` 深度优化标题

```
用户：选第一个，帮我写脚本
AI：调用 aigc-video-pipeline 阶段2...
```
