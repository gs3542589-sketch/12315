# Cover Designer - 封面设计工作流

## 技能说明
为短视频自动设计爆款封面，支持多平台尺寸，一键生成视觉冲击力强的封面图。

## 触发词
封面设计、视频封面、封面生成、制作封面、thumbnail

## 平台尺寸规范

| 平台 | 比例 | 尺寸(px) | 重点区域 |
|------|------|----------|----------|
| 抖音 | 9:16 | 1080×1920 | 中上部（避开底部互动区） |
| 小红书 | 3:4 | 1080×1440 | 中部（图文并茂） |
| 视频号 | 9:16 | 1080×1920 | 中上部 |
| B站 | 16:9 | 1920×1080 | 中部（标题安全区） |
| YouTube | 16:9 | 1280×720 | 中部 |

## 设计原则（爆款封面公式）

### 1. 视觉冲击力
- **对比色**：红vs绿、蓝vs橙、黄vs紫
- **高饱和度**：主角/产品突出
- **大字体**：标题占画面30-40%

### 2. 信息层次
```
【封面结构】
┌─────────────────┐
│   标题（大）      │ ← 30% 面积
│                 │
│   主体画面       │ ← 50% 面积
│   （人物/产品）   │
│                 │
│   副标题/标签    │ ← 20% 面积
└─────────────────┘
```

### 3. 爆款元素
| 元素 | 作用 | 示例 |
|------|------|------|
| 箭头/圆圈 | 引导视线 | → 看这里 |
| 数字 | 具体化 | "3个方法" |
| 表情包 | 情绪感染 | 😱🔥💪 |
| 前后对比 | 冲击力 | Before/After |
| 人物表情 | 代入感 | 惊讶/开心/疑惑 |

### 4. 字体规范
- **标题字**：粗体、无衬线、可读性强
- **字数限制**：
  - 抖音/视频号：≤8字
  - 小红书：≤12字
  - B站：≤15字
- **颜色**：白字+描边 / 亮色+阴影

## 工作流程

### Step 1: 收集信息
询问用户：
```
请提供封面设计信息：
1. 平台：抖音/小红书/视频号/B站/YouTube
2. 视频主题：[一句话描述]
3. 核心卖点：[最想传达的信息]
4. 风格偏好：
   - A. 高级感（简约/留白）
   - B. 冲击力（对比/夸张）
   - C. 种草风（精致/生活化）
   - D. 知识感（专业/可信）
5. 参考图（可选）：[上传参考图]
```

### Step 2: 生成方案
输出3个封面方案：

```
【封面方案A - 冲击力型】
┌─────────────────┐
│   😱 救命！      │
│                 │
│   [主体画面]     │
│   月入5万秘密    │
│                 │
│   → 3个方法      │
└─────────────────┘

设计要点：
- 红色背景+黄色大字
- 惊讶表情增强代入
- 箭头引导视线

【封面方案B - ...】
...
```

### Step 3: 图像生成
根据选定方案，生成封面图：

**生成方式**：
1. **调用 creative-toolkit**：使用5引擎路由生成
2. **DALL-E 3**：精准文字排版
3. **Midjourney**：艺术风格

**提示词模板**：
```
[平台] video thumbnail, [比例], [风格描述],
主标题"[标题文字]" in bold [颜色] font,
[主体描述],
[元素1], [元素2],
[氛围词], high quality, eye-catching,
safe zone for [平台] UI overlay
```

### Step 4: 优化调整
```
用户反馈：字太小了
AI调整：增大字体 → 重新生成

用户反馈：颜色太暗
AI调整：提高饱和度 → 重新生成
```

## 风格模板库

### 🔥 爆款风（抖音/视频号）
```
- 高饱和背景（红/橙/黄）
- 超大标题+描边
- 人物表情夸张
- 箭头/圆圈引导
```

### 🌸 种草风（小红书）
```
- 浅色/渐变背景
- 精致字体
- 产品特写
- 生活化场景
- 标签装饰
```

### 📚 知识风（B站）
```
- 深色背景
- 专业字体
- 图表/数据可视化
- 关键词提炼
```

## 与其他技能联动

- 接收 `aigc-video-pipeline` 的视频主题
- 调用 `creative-toolkit` 生成图像
- 输出封面供 `social-media-publish` 发布
