# AIGC Video Pipeline - 视频内容全流程生产线

## 技能说明
AIGC短视频一站式创作总控技能，串联选题→脚本→分镜→提示词→生成全流程。

## 触发词
AIGC创作、视频流水线、短视频制作、一键生成视频、内容生产

## 核心工作流

### 阶段1：选题策划 (Topic)
**输入**：赛道/关键词（可选）
**输出**：爆款选题 + 标题候选

**流程**：
1. 询问用户赛道（如：知识科普/情感故事/产品种草）
2. 调用热点分析（可接入bilibili-hot-monitor/xiaohongshu技能）
3. 输出3个选题方向 + 5个爆款标题候选

**爆款标题公式**（内置）：
- 二极管法：XX都可怕/说出来你可能不信
- 共情法：救命/逼自己看完/真心建议
- 数字法：我XX岁/XX个方法/XX天逆袭

---

### 阶段2：脚本创作 (Script)
**输入**：选题 + 标题
**输出**：完整分镜头脚本

**文案结构选择**：
| 结构 | 适用场景 | 公式 |
|------|----------|------|
| PREP | 知识科普 | 观点→理由→案例→重申 |
| 对比 | 产品种草 | 之前vs之后/别人vs我们 |
| FIRE | 情感故事 | 事实→影响→反应→结尾 |
| RIDE | 剧情向 | 共鸣→冲突→反转→升华 |

**输出格式**：
```
【分镜头脚本】
场景1：[画面描述] | [文案/对白] | 时长：X秒
场景2：...
...
总时长：30-60秒
```

---

### 阶段3：分镜设计 (Storyboard)
**输入**：分镜头脚本
**输出**：Seedance导演式分镜提示词

**万能公式**：
```
主体 + 动作过程 + 场景 + 光线 + 镜头语言 + 风格质感 + 画质 + 约束
```

**稳定词库**：
- 动作：缓慢、连贯、自然、慢动作
- 镜头：平稳跟拍、缓慢推进、固定机位
- 约束（必写）：
  1. 面部稳定不变形
  2. 手指不畸形
  3. 动作流畅不抽搐
  4. 画面稳定不闪烁
  5. 无字幕/水印/Logo

**输出格式**：
```
【分镜提示词】
场景1：
[完整Seedance提示词，含约束]

场景2：
...

预估生成参数：时长5-8秒/场景，新手友好
```

---

### 阶段4：视频生成 (Generate)
**输入**：分镜提示词
**输出**：视频片段

**模式选择**：
| 模式 | 适用场景 | 参数建议 |
|------|----------|----------|
| 文生视频 | 创意短片 | 提示词写细+约束完整 |
| 图生视频 | 风格统一 | 参考图定风格+提示词管动作 |
| 多模态 | 复杂场景 | 少喂喂准，每次≤3参考 |

**时长建议**：
- 新手：5-8秒/片段
- 有经验：10-15秒/片段
- 最终：多段拼接

---

### 阶段5：后期整合 (Post)
**输入**：视频片段
**输出**：完整视频

**处理项**：
- [ ] 音频：TTS配音 / 声音克隆(clonev) / 背景音乐
- [ ] 字幕：ai-subtitle-generator自动生成
- [ ] 剪辑：片段拼接 / 转场 / 调色

---

### 阶段6：发布分发 (Publish)
**输入**：成品视频
**输出**：多平台发布

**平台适配**：
| 平台 | 时长 | 比例 | 发布时间 |
|------|------|------|----------|
| 抖音 | ≤60s | 9:16 | 12:00/18:00/21:00 |
| 小红书 | ≤15min | 3:4 | 7:30/12:00/20:00 |
| 视频号 | ≤60s | 9:16 | 8:00/12:00/20:00 |

---

## 使用方式

### 完整流程
```
用户：帮我创作一个关于[主题]的短视频
AI：启动AIGC Video Pipeline → 依次执行6阶段 → 输出成品
```

### 单阶段调用
```
用户：只帮我写脚本
AI：执行阶段2 → 输出分镜头脚本

用户：只帮我生成分镜提示词
AI：执行阶段3 → 输出Seedance提示词
```

### 迭代优化
```
用户：这个分镜不够稳
AI：调整镜头词 + 增加约束 → 重新输出

用户：标题不够吸引
AI：换一个爆款公式 → 输出5个新标题
```

---

## 参数配置

### 默认风格
- 画质：4K, 高清, 细节丰富
- 色调：电影感, 自然光
- 帧率：24fps（电影）/ 30fps（流畅）

### 平台预设
- 抖音：快节奏、强冲突、前3秒抓眼球
- 小红书：精致、种草感、情绪共鸣
- B站：深度、知识感、信息密度高

---

## 技能依赖
- 内置：Seedance规范（MEMORY.md已固化）
- 可选接入：
  - xiaohongshu（选题热点）
  - openai-tts（配音）
  - clonev（声音克隆）
  - ai-subtitle-generator（字幕）
