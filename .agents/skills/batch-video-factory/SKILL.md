# Batch Video Factory - 批量视频生成工厂

## 技能说明
一键批量生成多条短视频，支持选题矩阵、脚本变体、风格切换，最大化内容产出效率。

## 触发词
批量生成、一键多发、批量视频、内容矩阵、量产视频

## 核心价值

| 维度 | 单条生产 | 批量生产 |
|------|----------|----------|
| 时间 | 30分钟/条 | 5分钟/条（均摊） |
| 成本 | 高 | 降低80% |
| 试错 | 1次机会 | 多版本测试 |
| 覆盖 | 单一受众 | 多受众矩阵 |

## 工作流程

### Step 1: 确定批量策略

询问用户：
```
请选择批量生成策略：

A. 同主题多角度
   - 1个主题 → 5个不同切入点
   - 例："副业赚钱" → 副业推荐/避坑/案例/工具/心态

B. 同选题多变体
   - 1个选题 → 3个不同表达方式
   - 例："3个方法" → 故事版/干货版/吐槽版

C. 系列内容包
   - 1个大主题 → 10条系列视频
   - 例："职场进阶" → 求职/面试/晋升/沟通/管理...

D. 多平台适配
   - 1条内容 → 3个平台版本
   - 例：抖音版/小红书版/B站版

请选择：A/B/C/D
```

### Step 2: 内容矩阵生成

#### 策略A：同主题多角度
```
输入主题：副业赚钱

【选题矩阵】
1. 副业推荐：2024最火的5个副业
2. 避坑指南：千万别做的3个副业
3. 真实案例：我靠副业月入3万的秘密
4. 工具分享：副业必备的10个神器
5. 心态建设：副业失败后我悟了

【输出】
每条包含：选题 + 标题 + 脚本大纲 + 分镜提示词
```

#### 策略B：同选题多变体
```
输入选题：3个副业赚钱方法

【变体矩阵】
1. 故事版：真实经历，情绪代入
2. 干货版：方法论，实操步骤
3. 吐槽版：反向切入，引发共鸣

【输出】
每个变体包含：完整脚本 + 分镜提示词
```

#### 策略C：系列内容包
```
输入大主题：职场进阶

【系列规划】
第1集：简历优化 - 让HR一眼看上你
第2集：面试技巧 - 3个问题定成败
第3集：谈薪资 - 多要30%的秘密
第4集：新人入职 - 前30天决定成败
第5集：职场沟通 - 向上管理的艺术
...

【输出】
每集包含：选题 + 标题 + 脚本 + 分镜
```

#### 策略D：多平台适配
```
输入内容：副业赚钱方法

【平台适配】
┌────────┬──────────────┬──────────────┐
│ 平台   │ 时长/形式     │ 内容调整      │
├────────┼──────────────┼──────────────┤
│ 抖音   │ 30s快节奏     │ 强化前3秒     │
│ 小红书 │ 60s精致感     │ 加生活化场景  │
│ B站    │ 3min深度版    │ 加数据分析    │
└────────┴──────────────┴──────────────┘
```

### Step 3: 批量生成执行

**执行模式**：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 串行 | 逐条生成，每条确认 | 高质量要求 |
| 并行 | 同时生成多条 | 效率优先 |
| 模板 | 基于模板快速填充 | 标准化内容 |

**进度展示**：
```
【批量生成进度】
✅ 第1条：脚本完成 → 分镜完成
⏳ 第2条：脚本生成中...
⏸️ 第3条：等待中
⏸️ 第4条：等待中
⏸️ 第5条：等待中

预计剩余时间：8分钟
```

### Step 4: 质量检查

**自动检查项**：
- [ ] 标题字数合规
- [ ] 脚本时长匹配
- [ ] 分镜提示词完整
- [ ] 约束条件齐全
- [ ] 无敏感词

**人工确认点**：
- 脚本逻辑确认
- 风格一致性确认
- 最终批量确认

### Step 5: 输出汇总

```
【批量生成完成报告】

📊 生成统计
- 总数量：5条
- 总时长：~4分钟内容
- 成功率：100%

📦 内容清单
1. ✅ 副业推荐 - 脚本+分镜已生成
2. ✅ 避坑指南 - 脚本+分镜已生成
3. ✅ 真实案例 - 脚本+分镜已生成
4. ✅ 工具分享 - 脚本+分镜已生成
5. ✅ 心态建设 - 脚本+分镜已生成

💾 导出选项
A. 全部导出为Markdown
B. 分条复制使用
C. 直接送入视频生成队列
```

## 参数配置

### 批量上限
- 单次最多：10条
- 建议范围：3-5条

### 差异化程度
| 级别 | 差异化 | 适用 |
|------|--------|------|
| 低 | 仅换标题 | 快速铺量 |
| 中 | 换角度/结构 | 多角度测试 |
| 高 | 完全重构 | 多受众覆盖 |

### 风格一致性
- 统一风格模板
- 统一约束条件
- 统一时长规格

## 效率提升技巧

### 1. 模板复用
- 固定开头/结尾模板
- 标准化分镜结构
- 可复用的B-roll清单

### 2. 批量操作
- 一次输入多个选题
- 批量生成提示词
- 批量导出

### 3. 自动化联动
- 热点自动触发批量生成
- 定时批量生产（每日/每周）
- 数据反馈自动优化
