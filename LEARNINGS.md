### 数据查询硬规则（新增）
1. **必须是最新数据**：只查询最近30天内的信息，超过30天的数据需要重新验证
2. **验证数据可靠性**：
   - 至少交叉验证2个以上不同来源
   - 检查数据来源是否权威可信
   - 标注数据来源和时间戳
3. **不確定则告知**：如果无法获取最新可靠数据，明确告知用户并建议替代方案
4. **记录数据时效**：所有引用数据必须标注获取日期和来源平台

### 数据查询流程（强制）
```
1. 确定查询主题和时间范围（最近30天）
2. 选择多个不同平台/来源交叉验证
3. 验证数据可靠性（来源权威性、点赞评论数、发布时间）
4. 标注数据来源和时间
5. 如无法获取可靠数据 → 明确告知用户并建议手动获取
```

## 2026-04-03（严重问题）

### 执行前思考不足

**问题表现**：
1. 用户要求生成视频提示词，我直接写了简短提示词，没有先用已学的seedance-shot-design技能
2. 用户指出"提示词质量差、缺少关键要素"，我才去读技能文档
3. 多次出现"先做再补"而不是"先学再做"

**根因分析**：
1. **急于交付**：想快速给用户结果，跳过了学习步骤
2. **技能遗忘**：安装了技能但没有在执行前主动调用
3. **流程缺失**：没有建立"先读技能→理解规范→按规范执行"的强制流程

**教训**：
- 安装技能≠掌握技能，必须在使用前主动读取SKILL.md
- 用户需求→先检索相关技能→读取技能规范→按规范执行
- 宁可慢一点，也要先学习再输出

**改进规则（强制）**：
1. **执行前检查**：收到任务后，先问自己"有没有相关技能？"
2. **技能优先**：如有相关技能，必须先读取SKILL.md再执行
3. **规范优先**：按技能规范执行，不凭记忆或经验
4. **慢即是快**：宁可多花时间学习，也不要输出低质量结果

**案例反思**：
- ❌ 错误：用户要视频提示词 → 直接写 → 被指出质量差 → 再去学技能
- ✅ 正确：用户要视频提示词 → 先读seedance-shot-design SKILL.md → 按五步工作流执行 → 输出专业提示词

## 视频提示词关键要素（2026-04-03深入学习）

### 六要素公式（强制）
```
[主体+外貌细节] + [动作+物理连贯性] + [场景环境] +
[视觉风格/物理光影] + [物理焦段+运镜] + [原生音效要求]
```

### 完整工作流程（2026-04-03更新）

```
第一步：生成参考图提示词
├── 关键人物提示词（外貌、服装、特征）
├── 关键道具提示词（外观、质感）
└── 关键场景提示词（环境、氛围、光影）

第二步：用参考图提示词生成参考图
├── 生成人物参考图
├── 生成道具参考图
└── 生成场景参考图

第三步：用参考图生成视频提示词
├── 引用人物参考图 @图片1
├── 引用道具参考图 @图片2
└── 引用场景参考图 @图片3

第四步：用参考图+视频提示词生成视频
```

**核心原则**：先有参考图保证一致性，再生成视频

### 参考图提示词要素
| 类型 | 要素 |
|------|------|
| 人物参考图 | 外貌特征、服装装备、配色方案、角度 |
| 道具参考图 | 外观造型、材质质感、配色方案 |
| 场景参考图 | 环境氛围、光影风格、建筑细节 |

### 1. 主体（Subject）
| 要素 | 说明 |
|------|------|
| 人物外貌 | 面部特征、服装、体型 |
| 身份细节 | 干员名称、武器装备 |
| 数量 | 单人/多人/群体 |

### 2. 动作（Action）
| 要素 | 说明 |
|------|------|
| 核心动作 | 一个时间段只描述一个动作 |
| 物理逻辑 | 重心转移、流体风阻、材质交互 |
| 动作连贯性 | 符合物理规律 |

### 3. 场景环境（Environment）
| 要素 | 说明 |
|------|------|
| 地点 | 地图名称、具体位置 |
| 氛围 | 室内/室外、天气、时间 |
| 细节 | 建筑风格、道具摆设 |

### 4. 视觉风格（Visual Style）
| 要素 | 说明 |
|------|------|
| 渲染引擎 | UnrealEngine5/Blender/Octane |
| 质感 | ARRI ALEXA/胶片颗粒/RAW质感 |
| 材质 | 皮肤/金属/玻璃/布料物理质感 |

### 5. 光影三层结构（强制）
```
光影：[光源词]（光源层），
[光行为词]（光行为层），
[色调公式]（色调层）。
```

| 层次 | 内容 | 示例 |
|------|------|------|
| 光源层 | 是什么光、从哪里来 | 爆炸橙红火光/冷蓝色战术光源 |
| 光行为层 | 光如何与材质/大气互动 | 体积光穿透/烟雾散射 |
| 色调层 | 整体冷暖和对比 | 冷蓝+暗红/橙红+灰色 |
