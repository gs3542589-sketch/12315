# 游戏二创AIGC全流程（完整版）

> 版本：1.0 | 更新：2026-04-05
> 适用：游戏角色搞笑整活/角色CP同人/影视换角/同人剧场
> 平台：抖音/快手/小红书/B站

---

## 一、核心定位

**目标**：15-40秒短视频爆款
**公式**：经典影视名场面魔改 + 游戏角色CP演绎 + 搞笑反差整活
**原则**：人设统一、节奏密集、梗点清晰、规避OOC/版权/限流

---

## 二、完整创作流程（9大阶段）

### 阶段1：需求锚定（10分钟）

#### 1.1 三要素锁定
| 要素 | 选项 |
|------|------|
| **内容形式** | 影视换角+CP / 整活剧场+CP / 经典复刻+整活 |
| **风格统一** | 沙雕魔性 / 甜宠搞笑 / 吐槽整活（不混搭） |
| **时长控制** | 抖音15-30s / B站1-2min / 小红书20-40s |

#### 1.2 爆款选题公式

**公式A：影视换角+CP**
```
国民级影视名场面 + 游戏高人气CP + 人设反差魔改
例：《狂飙名场面→王者双C互怼》
```

**公式B：整活剧场+CP**
```
游戏角色日常 + 网络热梗 + CP贴贴爆笑反转
例：《威虫×麦小鼠下班偶遇》
```

**公式C：经典复刻+整活**
```
影视名台词 + 游戏角色对口型 + 无厘头改编
例：《周星驰台词→游戏角色演绎》
```

#### 1.3 热门选题池
| 类型 | 来源 |
|------|------|
| **影视** | 周星驰系列、甄嬛传、狂飙、家有儿女、武林外传 |
| **游戏梗** | 角色名场面、版本新剧情、官方整活 |
| **网络梗** | 当期搞笑梗、CP磕糖梗、热梗二创 |

#### 1.4 红线避雷（必守）
- ❌ 不恶意抹黑角色、不拉踩其他CP
- ❌ 不涉及敏感影视/时政内容
- ❌ 不低俗恶搞、不扭曲官方核心人设
- ❌ 影视片段仅选喜剧/名场面，拒绝血腥/暴力
- ❌ 不完整搬运，单片段≤15s

### 阶段2：素材筹备（15分钟）

#### 2.1 素材分类表

| 类型 | 内容 | 用途 |
|------|------|------|
| **角色素材** | 表情特写、待机动作、技能动画 | 核心角色 |
| **CP素材** | 对视、贴贴、互动、并肩镜头 | 磕糖核心 |
| **整活素材** | 奇葩动作、BUG画面、失误镜头 | 笑点来源 |
| **影视素材** | 单片段≤15s，核心台词+关键镜头 | 换角素材 |
| **场景素材** | 游戏地图、基地门口、撤离点 | 背景环境 |

#### 2.2 版权三不原则
1. **不完整搬运**：仅魔改台词+换角色
2. **不盗成品**：只用官方录屏/自行录制
3. **BGM合规**：平台免费版权库，禁用付费OST

#### 2.3 素材归档规范
```
游戏名_素材类型_角色/CP名_版本
例：三角洲_角色图_威虫_呆萌
    三角洲_CP同框_威虫×麦小鼠_下班
    影视片段_狂飙名场面_高启强
```

### 阶段3：AI脚本生成（10分钟）

#### 3.1 黄金脚本结构

| 时段 | 功能 | 内容 |
|------|------|------|
| **0-3s** | 黄金钩子 | 影视名场面/CP对视/沙雕动作 |
| **3-25s** | 核心内容 | CP对手戏+台词魔改+搞笑行为 |
| **结尾** | 反转爆点 | 磕糖暴击/笑点反转/吐槽金句 |

#### 3.2 标准分镜格式

```markdown
镜头号|时长|画面内容|角色动作|台词|字幕|音效|BGM
1|0-3s|威虫在门口傻等|左右张望|无|无|轻快环境音|轻音乐
2|3-5s|麦小鼠背枪走出|步伐稳健|无|无|脚步声|无
3|5-7s|威虫疯狂挥手|跳起来挥手|无|无|心跳音效|甜宠BGM
...|...|...|...|...|...|...|...
```

#### 3.3 分镜生成提示词（AI用）

```
你是游戏二创编剧，风格【沙雕搞笑/甜宠/轻虐】，
为【游戏名】的【CP名：A×B】创作【时长】s小剧场，
剧情：【日常互动/名场面复刻/整活约会/影视换角】，
结构：3秒对视钩子→互动台词→反转/糖点，
输出标准分镜表，台词口语化无OOC，标注音效。
```

#### 3.4 短视频脚本模板

```markdown
# 【视频标题】

## 基本信息
- **时长**：【X】秒
- **平台**：【抖音/小红书/B站】
- **类型**：【搞笑整活/CP甜向/影视换角】

## 分镜脚本

### [0-3s] 黄金钩子
【画面】：角色特写/名场面开场
【动作】：具体动作描述
【台词】：钩子台词/无台词

### [3-Xs] 核心内容
【画面】：主体场景
【动作】：角色互动
【台词】：魔改台词/对口型

### [结尾] 爆点反转
【画面】：笑点/糖点
【字幕】：金句/吐槽
【音效】：强化音效

## 制作备注
- **BGM**：【风格】
- **字幕**：【样式】
- **特效**：【贴纸/转场】
```

### 阶段5：AI配音音效

#### 5.1 音色选型

| 类型 | 场景 | 推荐音色 |
|------|------|----------|
| **搞笑整活** | 沙雕角色 | 沙雕青年音、戏精女声 |
| **CP甜向** | 温柔场景 | 温柔少女音、低音炮男声 |
| **高冷角色** | 傲娇人设 | 高冷少女音、沉稳男声 |
| **热血战斗** | 战斗场景 | 热血少年音、御姐音 |

#### 5.2 阿里云CosyVoice配置

```bash
# 声音克隆流程
1. 提供参考音频URL（角色原声样本）
2. 调用voice-enrollment API克隆voice_id
3. 用voice_id进行TTS合成

# 模型选择
- cosyvoice-v3.5-plus（推荐，中文最优）
- cosyvoice-v3.5-flash（快速版）
```

#### 5.3 音效分层

| 层次 | 内容 | 音量 |
|------|------|------|
| **底层** | BGM | 15%-20% |
| **中层** | 角色台词/对白 | 100% |
| **顶层** | 心动/搞笑音效 | 50% |

#### 5.4 音效时间轴模板

```markdown
0-3s：环境音（轻快/紧张）
3-5s：脚步声/入场音效
5-7s：心跳/心动音效（CP向）
7-10s：温馨BGM渐入
10-13s：搞笑/尴尬音效
13-16s：[高潮音效]
16-20s：结尾音效/金句旁白
```

### 阶段6：剪辑精修

#### 6.1 剪辑技巧

**CP甜向**：
- 对视/贴贴：慢放0.5x + 柔光滤镜
- 转场：叠化，柔和过渡
- 特效：爱心/脸红/心动贴纸

**搞笑整活**：
- 沙雕动作：鬼畜重复3次，强化笑点
- 奇葩瞬间：慢放放大 + 吐槽字幕
- 节奏：1秒1镜，快切卡点

**影视换角**：
- 口型对齐：100%匹配原片节奏
- 镜头切换：原片→角色无缝衔接
- 音效：保留原片经典音效

#### 6.2 剪辑操作清单

| 步骤 | 操作 |
|------|------|
| **粗剪** | 按分镜8个镜头切割 |
| **调速** | 高潮慢放0.5x，整活快放1.1x |
| **调色** | 搞笑高饱和，CP暖调，统一色调 |
| **去瑕疵** | 裁剪AI畸形画面 |
