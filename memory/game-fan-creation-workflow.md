# 游戏二创AIGC全流程（2026-04-04）

> 来源：用户提供 | 适用：搞笑整活/角色CP/同人剧场/影视换角

---

## 核心定位
15-40s短视频爆款：经典影视名场面魔改+游戏角色CP演绎+搞笑反差整活

---

## 阶段1：选题定调

### 三要素
- 形式：影视片段换游戏角色+CP对手戏+搞笑反转
- 风格：沙雕魔性/甜宠搞笑/吐槽整活（不混搭）
- 时长：抖音15-30s，B站1-2min，小红书20-40s

### 爆款选题公式
1. 影视换角+CP：国民级影视名场面 + 游戏高人气CP + 人设反差魔改
2. 整活剧场+CP：游戏角色日常 + 网络热梗 + CP贴贴爆笑反转
3. 经典复刻+整活：影视名台词 + 游戏角色对口型 + 无厘头改编

### 热门选题池
- 影视：周星驰、甄嬛传、狂飙、家有儿女、武林外传
- 热梗：当期搞笑梗、CP磕糖梗、版本新剧情

### 红线避雷
- 不恶意抹黑角色、不拉踩CP、不涉及敏感内容
- 不低俗恶搞、不扭曲官方人设
- 影视片段仅选喜剧/名场面，拒绝血腥/暴力

---

## 阶段2：素材筹备

### 素材分类
1. **游戏角色素材**：表情特写、待机动作、技能动画、双人同框、羁绊剧情
2. **CP专属素材**：对视、贴贴、互动、并肩镜头
3. **整活素材**：奇葩动作、BUG画面、失误镜头
4. **影视素材**：单片段≤15s，核心台词+关键镜头
5. **辅助素材**：音效（问号/破防/心动）、贴纸、BGM

### 版权三不原则
1. 不完整搬运影视片段，仅魔改台词+换角色
2. 不盗用他人二创成品
3. BGM优先平台免费版权库

---

## 阶段3：脚本分镜

### 黄金脚本结构
1. 0-3s 钩子：影视名场面/CP对视/沙雕动作
2. 3-25s 主体：CP对手戏+台词魔改+整活
3. 结尾 反转：爆笑反差/磕糖暴击

### 分镜格式
```
镜头号|时长|画面内容|角色动作|台词|字幕|音效
1|0-3s|A怼脸特写|挑眉坏笑|哟，这不你吗？|哟，这不你吗？|心动音效
```

## 阶段4：视觉生成

### 方案1：AI角色替换
- 剪映/即梦/Runway
- 导入影视片段，上传游戏角色素材
- AI对口型+表情贴合

### 方案2：AI绘图
- Flux/SD/MJ/NanoBanana
- 提示词：游戏角色A和B，演绎XX影视名场面，搞笑沙雕

---

## 阶段5：配音音效

### 音色选型
- 搞笑整活：沙雕青年音、戏精女声
- CP甜向：温柔少女音、低音炮男声

### 音效分层
1. 底层：BGM（音量15%-20%）
2. 中层：角色台词（100%）
3. 上层：整活/心动音效

---

## 阶段6：剪辑

### 剪辑技巧
- CP向：对视慢放0.5x+柔光，贴贴加爱心特效
- 整活向：鬼畜重复3次，慢放放大，1秒1镜

### 操作流程
1. 粗剪：按分镜切割
2. 调速：高潮慢放，整活快放
3. 调色：搞笑高饱和，CP暖调
4. 去瑕疵：裁剪畸形画面

---

## 阶段7：包装发布

### 封面制作
- 主标题：影视+游戏
- 副标题：梗点
- 角色大头贴居中

3. 整活反转：《谁懂啊！角色演绎名场面太适配了》

### 标签组合

## 极简SOP（1小时出片）

```
1. 定选题：选热门影视梗+游戏CP
2. 备素材：截影视片段+录游戏角色镜头
3. AI生成：脚本分镜+角色换脸/绘图
4. 配音频：AI配音+卡点音效+BGM
5. 精剪辑：卡点，调速，加特效
6. 做包装：封面+字幕+水印
7. 合规查+发布+评论区运营
```
