> 替代：原7阶段42步 → **4阶段12步**

## 二、阶段1：定版准备（防崩脸核心）

### 1.1 选题定调（1分钟）

**快速填表**：
| 项目 | 填写 |
|------|------|
| 主题 | ___ |
| 受众 | ___ |
| 风格 | ___ |
| 时长 | 15s/30s/60s |
| 平台 | 抖音/小红书/B站 |
| 情绪 | 治愈/紧张/搞笑/感动 |

### 1.2 定版素材（必须先做！）

**这是防崩脸的核心，很多人出问题就因为跳过这步**

| 类型 | 内容 | 工具 | 检查点 |
|------|------|------|--------|
| 人物三视图 | 正面+侧面+背面 | NanoBanana | 五官/发型/服饰/配饰完全一致 |
| 关键道具 | 核心物品特写 | NanoBanana | 细节清晰 |
| 核心场景 | 主要环境 | NanoBanana | 光影/色调满意 |

**定版标准**：反复调整直到**完全满意**，后续所有生成都以此为基准。

### 1.3 提示词模板（套用7大公式）

**万能公式**：
```
核心叙事主体 + 精准动作指令 + 镜头语言 + 光影氛围 + 画面风格 + 画质参数 + 一致性锁定
```

**模板示例**（可直接套用）：
```
【主体】二十岁古装男子，剑眉星目，墨色长发束起，藏青锦袍，腰佩白玉剑，神情冷峻
【动作】沿青石路缓步前行，目光锁定前方古宅，右手缓抬握住剑柄
【镜头】中景→近景缓慢推进，平视，固定机位
【光影】侧逆光勾勒轮廓，暖黄色调，电影质感
【风格】古风武侠，写实风格，8K画质
【约束】面部稳定不变形，手指不畸形，动作流畅无抽搐，画面稳定不闪烁
【一致性】人物形象必须完全参考@定版图，五官服饰配饰体态完全保持一致
```

## 三、阶段2：批量生成（效率最大化）

### 2.1 分镜拆解（15秒/镜）

**动作描写原则**：写「过程」而非「结果」
- ❌ 主角走在路上
- ✅ 主角沿青石路缓步前行，目光落在前方古宅，手轻轻握住腰间佩剑

**分镜衔接**：前镜结尾动作 = 后镜起始动作

**15秒万能分镜**：

| 时段 | 内容 | 镜头 |
|------|------|------|
| 0-3秒 | 黄金开场：核心卖点特写 | 快速推近 |
| 3-8秒 | 细节展示：360度环绕 | 匀速环绕 |
| 8-12秒 | 场景植入 | 缓慢拉远 |
| 12-15秒 | 收尾引导 | 固定镜头 |

### 2.2 批量出图/视频

**效率技巧**：
- 同提示词生成 **5-10版**，选最优
- 九宫格成片：NanoBanana生成分镜图 → Seedance一键成片
- 一致性锁链：前镜定稿 = 后镜参考

### 2.3 首轮粗筛

**淘汰标准**：
- ❌ 崩脸/五官变形
- ❌ 服饰跑偏
- ❌ 动作不符分镜
- ❌ 场景不一致
- ❌ 画质模糊/卡顿

## 四、阶段3：质量把控（检查点机制）

### 3.1 逐帧检查清单

**每镜必查**（打勾才算过）：
- [ ] 五官无变形
- [ ] 手指无畸形/多指/缺指
- [ ] 服饰无跑偏
- [ ] 动作符合分镜设定
- [ ] 场景与定版一致
- [ ] 画面无闪烁/卡顿
- [ ] 无字幕/水印/Logo

### 3.2 精修整合

**剪映流水线**：
```
导入片段 → 调速/转场 → 配音 → 字幕 → BGM → 调色 → 导出
```

**工具对应**：
| 环节 | 工具 |
|------|------|
| 配音 | openai-tts / 阿里CosyVoice |
| 字幕 | ai-subtitle-generator |
| BGM | music-cog |
| 剪辑 | 剪映/pyJianYingDraft |

### 3.3 合规审核

**必查项**：
- [ ] 肖像权（人物脸/身体）
- [ ] 音乐版权
- [ ] 字体版权
- [ ] 违禁词检测
- [ ] 事实核查

## 五、阶段4：发布复盘（爆款沉淀）

### 4.1 平台适配

| 平台 | 标题 | 封面 | 标签 | 话题 |
|------|------|------|------|------|
| 抖音 | 15字内悬念 | 9:16高对比 | 3-5个 | 2个热门 |
| 小红书 | 18字情绪词 | 3:4精致感 | 5-8个 | 1个垂直 |
| B站 | 20字信息量 | 16:9电影感 | 5个 | UP主话题 |

### 4.2 发布监测

**核心指标**：
| 指标 | 公式 | 健康值 |
|------|------|--------|
| 完播率 | 完播数/播放数 | >30% |
| 互动率 | (赞+评+转)/播放 | >5% |
| 转化率 | 点击/播放 | >2% |

### 4.3 复盘沉淀

**爆款公式提炼**：
```
爆款 = 选题热度 × 开场3秒 × 内容价值 × 情绪共鸣 × 发布时机
```

**沉淀到提示词库**：
- 成功的提示词 → 标注效果 → 下次复用
- 失败的提示词 → 标注问题 → 避坑

## 六、一键流水线（工具链整合）

### 完整流水线命令

```bash
# 1. 选题 → 定版素材
image-cog "人物三视图：[描述]" --count 3

# 2. 分镜 → 视频
seedance-video-generation --reference @定版图 --prompt "[7大公式]"

# 3. 配音
openai-tts --text "[旁白]" --voice alloy

# 4. 字幕
ai-subtitle-generator --video output.mp4 --languages zh

# 5. BGM
music-cog --style "[风格]" --duration 15

# 6. 发布
social-media-publish --platform douyin --draft
```

### 技能速查表

| 阶段 | 技能 | 用途 |
|------|------|------|
| 定版素材 | image-cog | 人物/场景/道具图 |
| 分镜脚本 | video-script-generator | 自动生成脚本 |
| 分镜表格 | script-to-storyboard | 脚本→分镜表 |
| 视频生成 | Seedance 2.0 | 图生视频 |
| 配音 | openai-tts | 文字转语音 |
| 字幕 | ai-subtitle-generator | 自动字幕 |
| BGM | music-cog | AI音乐 |
| 发布 | social-media-publish | 多平台发布 |

## 十、日常执行版（每天直接套用）

### 极简SOP（8步）
```
1. 选题定调（1分钟）
2. 定版素材（反复调整到满意）
3. 写提示词（套模板）
4. 批量生成（5-10版）
5. 逐帧检查（打勾）
6. 精修整合（剪映）
7. 合规审核
8. 发布复盘
```

### 时间分配建议
| 阶段 | 时间 | 占比 |
|------|------|------|
| 定版准备 | 30% | 最重要，防崩脸 |
| 批量生成 | 20% | 效率环节 |
| 质量把控 | 30% | 精修出精品 |
| 发布复盘 | 20% | 数据驱动迭代 |
