# Part 1：底层方法论

**核心认知**：Seedance 2.0 不是关键词堆砌工具，而是"听话的执行导演"。

- 不吃零散的关键词堆砌
- 只吃完整的、有时间线、有画面逻辑、有细节指令的导演式叙事文本

**正确姿势**：把自己当导演，像对着摄影师说话一样，清清楚楚告诉它：每一秒要拍什么、怎么拍、用什么光、什么节奏、什么感觉。

## 抽象词「翻译表」

| 抽象词 | 具体拆解 |
|--------|---------|
| **高级感** | 低饱和度莫兰迪色系 + 45度侧逆光勾勒主体轮廓 + 浅景深虚化背景 + 画面留白占比60% + 匀速慢镜头运镜 |
| **氛围感** | 暖黄色柔光 + 轻微胶片颗粒质感 + 前景虚化遮挡 + 慢门动态模糊 + 环境音同步 |
| **电影感** | 2.35:1宽画幅 + 明暗对比强烈的伦勃朗光 + 镜头缓慢推拉 + 低饱和影调 + 前景纵深构图 |
| **治愈感** | 明亮柔和的自然光 + 高通透浅色系画面 + 微距特写细节 + 缓慢平移运镜 + 白噪音背景音 |

## 掐着15秒写分镜

**黄金3秒定律**：开头3秒必须抓住用户眼球。
**分镜数量**：15秒视频，分镜控制在 3-5段，每段至少2-3秒，少而精。

### 万能分镜模板（好物/产品赛道）

| 时段 | 内容 | 镜头操作 |
|------|------|---------|
| 0-3秒 | 黄金开场：核心卖点特写 | 快速推近，直击痛点 |
| 3-8秒 | 细节展示：360度环绕 | 匀速环绕，侧逆光 |
| 8-12秒 | 场景植入 | 缓慢拉远，放入真实场景 |
| 12-15秒 | 收尾引导 | 固定镜头，搭配slogan |

### 万能分镜模板（氛围/情绪赛道）

| 时段 | 内容 | 镜头操作 |
|------|------|---------|
| 0-3秒 | 开场定调：大场景空镜 | 缓慢运镜，定好色调氛围 |
| 3-7秒 | 细节特写 | 微距拍环境细节 |
| 7-12秒 | 主体植入 | 镜头拉远，带出主体 |
| 12-15秒 | 收尾留白 | 固定长镜头，慢慢暗下 |

# Part 2：提示词公式

## 2A：S-A-C-S-C 五步提问法

| 步骤 | 要素 | 说明 |
|------|------|------|
| **S** | Subject 锚定主体 | 提供人物/产品参考，@引用参考图，拒绝模糊 |
| **A** | Action 定义动作 | 清晰描述行为，聚焦物理状态变化，用英文动词 |
| **C** | Camera 规范运镜 | 专业摄影术语，明确景别+运镜方式，不用主观词 |
| **S** | Style 设定风格 | 统一光线+画质+艺术风格，用具体参数替代"高级感" |
| **C** | Constraints 添加约束 | 提前规避AI常犯错误（变形/穿帮/卡顿） |

## 2B：7大电影级公式（尤道子）

```
核心叙事主体 + 精准动作指令 + 镜头语言 + 光影氛围 + 画面风格 + 画质参数 + 一致性锁定
```

| 模块 | 内容要点 | 权重位置 |
|------|---------|---------|
| **核心叙事主体** | 年龄/性别/五官/发型/服饰/配饰/体态/情绪，全部写清 | 放最前面（权重最高） |
| **精准动作指令** | 起始→过程→结尾，写清幅度/速度/情绪配合 | 接主体之后 |
| **镜头语言** | 景别（远/全/中/近/特）+ 运镜（固定/推/拉/摇/跟/移）+ 视角（平/仰/俯） | 第三位 |
| **光影氛围** | 光源类型 + 光效（柔/硬/逆）+ 色调（暖/冷）+ 整体氛围 | 第四位 |
| **画面风格** | 统一光线+画质+艺术风格，用参数替代"高级感" | 第五位 |
| **画质参数** | 8K/4K + 写实/动漫风 + 相机焦距 | 第六位 |
| **一致性锁定** | 必须强调以参考图为准，防止崩脸 | 每个分镜都要写 |

### 7大公式示例

❌ 错误（堆砌）：
> 古装男子，微笑，走路，帅

✅ 正确：
> 二十岁左右的古装男子，剑眉星目，高挺鼻梁，墨色长发束起，身着藏青色锦袍，腰间佩戴白玉佩剑，身形挺拔，神情冷峻；主角沿着青石板路缓步向前行走，步伐平稳，目光始终锁定前方古宅大门，右手缓缓抬起握住腰间佩剑剑柄，动作舒缓有力，神情保持警惕；中景侧跟镜头，平视视角；夕阳侧逆光勾勒轮廓，暗部保留细节，整体氛围冷峻肃杀；电影写实风格，8K画质；人物形象必须完全参考输入的定版图，五官、服饰、配饰、体态完全保持一致，不能有任何改动。

# Part 3：进阶技巧

## 万能调整公式（出问题后不用重写）

```
保留XX（满意的内容） + 把XX改成XX（要改的内容）
```

| 问题 | 调整方法 |
|------|---------|
| 运镜太快/卡顿 | 保留整体色调、画面内容和分镜节奏，把所有镜头的运镜改为匀速缓慢运动，镜头切换时加入0.5秒的淡入淡出转场 |
| 画面跑偏/主体不对 | 保留整体的运镜、光影和分镜，严格以@图片1产品主体为核心，删除背景中多余杂物，主体始终在画面中心 |
| 质感不够/色调不对 | 保留所有分镜、运镜和画面内容，把冷色调换成暖黄色柔光，画面增加轻微胶片颗粒质感 |

## 负面提示词（神级功能，每次必加）

```
画面模糊、卡顿、人物变形、手部畸形、色彩溢出、
穿帮镜头、逻辑混乱、画面抖动、主体偏移、透视畸变
```

## 素材参考标准写法

✅ 正确（明确标注用途）：
> @图片1 作为人物主体参考，严格保留五官、发型、服饰特征；@图片2 作为场景背景参考，保留环境色调；@视频1 的运镜方式完全复刻，推进速度保持一致

❌ 错误（只贴不说明）：
> @图片1@视频1@音频1 古装场景

# Part 4：参数调优 + 动作词库

## Seedance 2.0 参数速查表

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 视频时长 | 5-10秒 | 新手不超过8秒 |
| 画幅比例 | 16:9(电影)/9:16(短视频)/1:1(方图) | 按发布平台选 |
| 帧率 | 24fps(电影)/30fps(流畅)/60fps(慢动作) | 按风格选 |
| Seed种子 | 固定数值 | 固定可复现，留空=创意 |
| 创意度 | 0.3-0.6 | 太高容易跑偏 |
| 运动强度 | 中等 | 高=快节奏打斗，低=慢镜头 |
| 参考强度 | 0.6-0.85 | 参考图/视频的权重 |

## @多模态参考语法

- 最多：9张图 + 3段视频(≤15秒) + 3段音频(≤15秒)
- 编号：按上传顺序从1开始

| 标签 | 用途 |
|------|------|
| `@Image1` | 参考图片1的外观/服装/主体 |
| `@Image2` | 参考图片2的背景/道具 |
| `@Video1` | 延续视频1的动作和运镜 |
| `@Audio1` | 口型同步音频1 |

## 稳定 vs 不稳定动作词

### 优先用（稳定）
```
缓慢、连贯、自然、慢动作、平稳跟拍、脚步轻移、平稳横移、
微微低头、轻微拉远、轻微环绕、轻轻抬手、缓慢转身
```

### 避免用（不稳定）
```
夸张、高速、复杂多人互动、大幅度扭曲、剧烈、快速横摇
```

### 动作描述原则
- 写「过程」而非「结果」
- ❌ `跳舞` → ✅ `随着节拍缓慢转身，手臂缓缓抬起划出弧线`
- ❌ `微笑` → ✅ `先是怔住，瞳孔微颤，而后笑意从嘴角漾开，染亮整双眼眸`

## 5B：核心10组详解

### 第1组：希区柯克滑动变焦（精神压迫）
**即梦模式**：`希区柯克推进`，速度0.7，FOV 50mm→24mm
```
轴向推轨变焦运镜，希区柯克滑动变焦，
主体画幅全程恒定，背景透视渐进畸变
```
**适合**：危机降临、精神冲击、紧张对峙

### 第2组：纵深渐进式定焦推轨（情绪聚焦）
**即梦模式**：`自定义推镜`，速度0.5缓入缓出，FOV固定35mm
```
纵深渐进式定焦推轨运镜，定焦镜头无透视畸变，
画面焦点全程锁定主体面部，匀速直线推进
```
**适合**：情绪递进、眼神特写、关键道具揭示

### 第3组：全域展开式定焦拉轨（全景揭示）

# Part 6：景别套路

## 吵架戏万能景别
**逻辑**：中 → 近 → 特 → 中 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 中景 | 开场冲突，交代两人关系、距离、环境 | 让观众知道谁在吵、在哪吵 |
| 近景 | 情绪升级，切到面部表情 | 放大语气、眼神、微表情 |
| 特写/大特写 | 激烈爆发：眼睛、嘴唇、颤抖的手、眼泪 | 把情绪怼到观众脸上 |

| 全景/远景 | 冷战收尾 | 两人离得远，画面空旷、疏离 |

## 氛围感走路景别（雨天/情绪向）
**逻辑**：远 → 中 → 近 → 特 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 远景 | 人物从远处走来，撑着伞，小小的身影 | 交代场景氛围 |
| 中景 | 正面/侧面走路，伞遮住半张脸，脚步匀速 | 看清楚人物+动作+环境 |
| 近景 | 拍到胸口以上，看表情、眼神 | 情绪开始进来 |
| 特写 | 鞋子踩进水洼，溅起水花 | 穿插细节，增加味道 |
| 远景 | 慢慢走远，背影变小 | 收尾留白 |

## 枪战反转分镜
**逻辑**：铺垫 → 行动 → 反转收尾

| 阶段 | 内容 |
|------|------|
| 铺垫 | BOSS手持加特林疯狂输出；主角躲掩体后快速换弹，神情紧张；好友在另一掩体后比手势，吸引火力 |

## 正反打镜头（180度轴线规则）

### 基础正反打（纯对话，聚焦情绪）

| 镜头 | 视角 | 画面 | 作用 |
|------|------|------|------|
| 正拍 | 从A的视角拍B | B的脸/上半身 | 让观众看到B在说话 |
| 反打 | 从B的视角拍A | A的脸/上半身 | 让观众看到A的反应 |

### 过肩正反打（最常用，锚定空间）

| 镜头 | 视角 | 画面 | 作用 |
|------|------|------|------|
| 过肩正拍 | 越过A的肩膀拍B | B的脸清晰，A的肩膀/后脑勺在画面一侧 | 锚定两人空间位置 |

**核心原则**：所有机位必须在两人连线的同一侧，**绝对不能越轴**

# Part 8：完整工作流（小说/故事→视频）

## 4步制作流程

### Step 1：分镜拆解（DeepSeek/豆包）
- 单分镜固定时长：**15秒**（即梦最优生成时长）
- 每个分镜包含4要素：核心情节动作 + 出场人物 + 场景环境 + 关键道具
- 动作描写：起始→过程→结尾，不写笼统模糊内容
- 分镜衔接：前一个分镜结尾动作 = 后一个分镜起始动作

### Step 2：定版素材制作（文生图）
**防崩脸的核心！跳过这步出问题概率90%**

| 类型 | 内容 |
|------|------|
| 人物三视图 | 正面+侧面+背面，全部统一五官/发型/服饰/配饰/体态 |
| 关键道具 | 佩剑、玉佩、核心物品 |
| 核心场景 | 古宅、山林、朝堂大殿等 |

生成后**反复调整直到完全满意**，后续所有视频生成都以定版图为基准。

### Step 3：逐镜生成（Seedance 2.0 图生视频）
- 以对应的定版图作为起始参考
- 逐镜生成 15 秒，**逐帧检查**：崩脸/服饰跑偏/动作不符/场景不一致
- **一致性锁链**：前一个分镜的定稿视频 = 后一个分镜的一致性参考
- 不满意就**调整提示词重新生成**，不要批量生产后返工

### Step 4：剪映拼接
- 按叙事顺序导入所有15秒片段
- 调整播放速度、转场效果
- 配 BGM + 人声配音 + 音效
- 统一色调，导出成片

## 实操检查清单（每个分镜生成前必查）

- [ ] 人物三视图已生成且满意
- [ ] 关键道具定版图已生成
- [ ] 核心场景定版图已生成
- [ ] 提示词包含7大模块
- [ ] 动作描写是"过程"而非"结果"
- [ ] 一致性锁定指令已写入
- [ ] 负面提示词已添加
- [ ] 景别+运镜+视角已明确

# Part 10：核心总结

## 写好Seedance提示词的本质
- 站在AI角度，把创意拆成清晰、具体、有逻辑的执行指令
- 不用华丽辞藻，不用堆关键词
- 逻辑顺了，AI就能1:1还原你的想法

**不管是AI工具还是实拍，核心差距不是工具，而是「把创意落地成精准指令」的能力。**
