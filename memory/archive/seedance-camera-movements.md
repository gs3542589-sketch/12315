## 二、核心10组详解

### 第1组：希区柯克滑动变焦（精神压迫）
**即梦模式**：`希区柯克推进`，速度0.7，FOV 50mm→24mm
```
轴向推轨变焦运镜，希区柯克滑动变焦，
主体画幅全程恒定，背景透视渐进畸变
```
**适合**：危机降临、精神冲击、紧张对峙

### 第2组：纵深渐进式定焦推轨（情绪聚焦）
**即梦模式**：`自定义推镜`，速度0.5缓入缓出，FOV固定35mm
```
纵深渐进式定焦推轨运镜，定焦镜头无透视畸变，
画面焦点全程锁定主体面部，匀速直线推进
```
**适合**：情绪递进、眼神特写、关键道具揭示

### 第3组：全域展开式定焦拉轨（全景揭示）
**即梦模式**：`自定义拉镜`，速度0.6，FOV固定40mm
```
全域展开式定焦拉轨运镜，定焦镜头无畸变，
从主体细节匀速拉远至全景，焦点清晰
```
**适合**：环境全貌揭示、剧情反转、宏大场面

### 第4组：轨道式平行环绕（角色高光）
**即梦模式**：`环绕运镜`，顺时针，速度0.4，半径3m，FOV 50mm
```
轨道式水平平行环绕运镜，摄影机与主体距离恒定，
360度匀速环绕，主体全程锁定画面中心
```
**适合**：角色登场、高光时刻、仪式感展示

### 第5组：垂直爬升式升降（史诗开场）
**即梦模式**：`升降运镜`，向上，速度0.55，高度1.7m→15m，FOV 45mm
```
垂直爬升式升降运镜，摄影机匀速垂直向上爬升，
从人物中景平稳过渡到高空大全景
```
**适合**：开场定场、史诗感、处境烘托

### 第6组：垂直俯冲式升降（悬念降临）
**即梦模式**：`升降运镜`，向下，速度0.65，高度20m→1.2m，FOV 50mm
```
垂直俯冲式升降运镜，摄影机从高空匀速垂直向下俯冲，
从上帝视角全景平稳过渡到主体特写
```
**适合**：危机降临、宿命感、关键线索揭示

### 第7组：水平缓速横摇（环境扫描）
**即梦模式**：`摇镜运镜`，从左至右，速度0.3，角度90°，FOV 35mm
```
水平缓速横摇运镜，机位固定不动，仅水平匀速摇镜，
画面全程清晰无抖动，完整扫描环境全貌
```
**适合**：场景环境交代、长镜头叙事、群像戏

### 第8组：急加速甩镜（节奏爆发）
**即梦模式**：`摇镜运镜`，从左至右，速度1.0极速，角度120°，FOV 24mm
```
急加速甩镜运镜，从左至右极速横甩，
甩动过程带动态模糊拖影，落幅精准锁定新主体
```
**适合**：动作戏转场、节奏卡点、情绪爆发

### 第9组：侧方跟移运镜（动作戏跟随）
**即梦模式**：`跟移运镜`，速度0.8，侧方3m，FOV 50mm
```
侧方跟移运镜，摄影机与奔跑的人物保持平行同步位移，
侧方视角，全程锁定人物于画面左侧三分之一处
```
**适合**：人物奔跑跟随、动作戏展现、运动连贯性

### 第10组：POV第一视角主观（沉浸代入）
**即梦模式**：`自定义运镜`，关闭主体锁定，手持微晃0.2，高度1.7m，FOV 50mm
```
POV第一视角主观运镜，完全模拟人眼视角，
模拟人物行走时的头部微晃与视线转移
```
**适合**：角色主观体验、悬疑探索、沉浸感第一人称

| FOV | 焦段 | 适合场景 |

| 12-16mm | 超广角 | 透视畸变、空间感、FPV |
| 24-35mm | 广角/标准 | 自然透视、环境扫描、跟移 |
| 40-50mm | 标准焦 | 人眼视角、人物、叙事 |

| 200mm+ | 超长焦 | 极压缩感、宿命感 |

| 慢速 | 0.3-0.5 | 叙事、氛围、标准 |
| 中速 | 0.5-0.7 | 流畅、标准叙事 |
| 快速 | 0.7-0.9 | 动感、节奏 |
| 极速 | 1.0 | 甩镜、子弹时间、爆发 |
