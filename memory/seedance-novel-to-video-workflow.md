## 一、完整制作流程（4步）

### Step 1：小说分镜拆解（DeepSeek/豆包）
- **单分镜固定时长**：15秒（即梦最优生成时长）
- **每个分镜必须包含4要素**：
  1. 核心情节动作
  2. 出场人物
  3. 场景环境
  4. 关键道具
- **动作描写原则**：精准到动作的起始→过程→结尾，不写笼统模糊内容
  - ❌ 主角走在路上
  - ✅ 主角身着藏青色锦袍，沿着青石板路缓步前行，目光落在前方古宅大门，手部轻轻握住腰间佩剑
- **分镜衔接**：前一个分镜的结尾动作 = 后一个分镜的起始动作，保证叙事连贯

### Step 2：定版素材制作（文生图）
**这是防崩脸的核心！很多人出问题就因为跳过了这步**

必须提前生成：
| 类型 | 内容 | 比例 |
|------|------|------|
| 人物三视图 | 正面+侧面+背面，全部统一五官/发型/服饰/配饰/体态 | 16:9 或 9:16 |
| 关键道具 | 佩剑、玉佩、核心物品 | 同上 |
| 核心场景 | 古宅、山林、朝堂大殿等 | 同上 |

生成后**反复调整直到完全满意**，后续所有视频生成都以定版图为基准。

### Step 3：逐镜生成（Seedance 2.0 图生视频）
- 以**对应的定版图**作为起始参考
- 逐镜生成 15 秒，**逐帧检查**：
  - 人物有没有崩脸
  - 服饰有没有跑偏
  - 动作是否符合分镜设定
  - 场景是否和定版场景一致
- **一致性锁链**：前一个分镜的定稿视频 = 后一个分镜的一致性参考
- 不满意就**调整提示词重新生成**，不要批量生产后返工

### Step 4：剪映拼接
- 按叙事顺序导入所有15秒片段
- 调整播放速度、转场效果
- 配 BGM + 人声配音 + 音效
- 统一色调，导出成片

## 二、7大电影级提示词公式（适配即梦Seedance）

```
核心叙事主体 + 精准动作指令 + 镜头语言 + 光影氛围 + 画面风格 + 画质参数 + 一致性锁定
```

### 模块1：核心叙事主体（权重最高，放提示词最前面）
**必须包含（人物）**：
- 年龄、性别
- 五官特点
- 发型样式
- 服饰材质/颜色/款式
- 配饰细节
- 体态
- 当前情绪状态

**示例**（反面→正面）：
- ❌ 古装男子
- ✅ 二十岁左右的古装男子，剑眉星目，高挺鼻梁，墨色长发束起，身着藏青色锦袍，腰间佩戴白玉佩剑，身形挺拔，神情冷峻

### 模块2：精准动作指令（保证叙事性）
- 必须**连续、可落地**
- 写清：起始状态 + 过程 + 结尾状态
- 写清：动作幅度 + 速度 + 情绪配合
- 时长要符合15秒限制，不要写太复杂的动作

**示例**：
- ❌ 主角走路
- ✅ 主角沿着青石板路缓步向前行走，步伐平稳，目光始终锁定前方古宅大门，右手缓缓抬起，握住腰间佩剑剑柄，动作舒缓有力，神情保持警惕

### 模块3：镜头语言（电影级质感核心）
| 子模块 | 选项 |
|--------|------|
| 景别 | 远景/全景/中景/近景/特写 |
| 运镜方式 | 固定镜头/推/拉/摇/跟/移 |
| 拍摄视角 | 平视/仰视/俯视/侧方 |

### 模块4：光影氛围
- 光源类型（自然光/台灯/蜡烛等）
- 光效（柔光/硬光/逆光等）
- 色调（暖黄/冷蓝/青橙等）
- 整体氛围（压抑/温馨/紧张等）

### 模块5：画面风格
- 统一光线+画质+艺术风格
- 用具体参数替代"高级感"

### 模块6：画质参数约束
- 8K/4K
- 写实/电影感/动漫风
- 相机型号+焦距+光圈

### 模块7：一致性锁定指令
**这是防崩的关键！每个分镜都要强调**
```
人物形象必须完全参考输入的定版图，
五官、服饰、配饰、体态完全保持一致，
不能有任何改动。
```
每个分镜的提示词里都要重复这条，锁死一致性。

## 三、与现有技巧的整合

### 已知强项（memory/seedance2-advanced-techniques.md）
- S-A-C-S-C 五步提问法
- 九宫格/十二宫格一键成片
- 负面提示词包
- 参数调优表（时长/帧率/创意度）
- 景别万能套路（吵架戏/走路镜头/枪战反转）
- 动作词库（稳定vs不稳定）

### 尤道子新补充的增量
1. **小说改视频完整流水线**（Step1-4全流程）
2. **定版素材先行的防崩意识**
3. **7大公式**（更细化的提示词结构，比S-A-C-S-C多了光影+画质+一致性锁定）
4. **动作描写原则：写过程而非写结果**

### 两者融合建议
- **选题阶段**：用尤道子7大公式写分镜提示词（更结构化）
- **九宫格成片**：用尤道子首发九宫格提示词模板（快速批量）
- **单镜精细打磨**：结合 S-A-C-S-C + 稳定动作词库
- **负面提示词**：始终加上，防止AI翻车
