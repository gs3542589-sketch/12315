## 一、Seedance 2.0 万能调整公式

### 核心公式
> 保留XX（满意的内容） + 把XX改成XX（要改的内容）

### 高频案例

| 问题 | 调整方法 |
|------|---------|
| 运镜太快/卡顿 | 保留整体色调、画面内容和分镜节奏，把所有镜头的运镜改为匀速缓慢运动，镜头切换时加入0.5秒的淡入淡出转场 |
| 画面跑偏/主体不对 | 保留整体的运镜、光影和分镜，严格以@图片1产品主体为核心，删除背景中多余杂物，主体始终在画面中心 |
| 质感不够/色调不对 | 保留所有分镜、运镜和画面内容，把冷色调换成暖黄色柔光，画面增加轻微胶片颗粒质感 |

## 三、素材正确用法

### ✅ 正确用法（明确标注用途）
> @图片1 作为蜜桃果茶的主体参考，严格保留杯子的造型、logo和果肉的颜色；@视频1 的360度匀速环绕运镜完全复刻，运镜速度保持一致；@音频1 的轻音乐作为背景音，镜头切换卡点音乐的鼓点，每4秒切换一次镜头。

### ❌ 错误用法（只贴不说明）
> @图片1@视频1@音频1 果茶展示。

## 四、分镜景别万能套路（实拍/AI通用）

### 1. 吵架戏万能景别
**逻辑**：中 → 近 → 特 → 中 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 中景 | 开场冲突，交代两人关系、距离、环境 | 让观众知道谁在吵、在哪吵 |
| 近景 | 情绪升级，切到面部表情 | 放大语气、眼神、微表情 |
| 特写/大特写 | 激烈爆发：眼睛、嘴唇、颤抖的手、眼泪 | 把情绪怼到观众脸上 |
| 中景 | 崩溃/哭（情绪崩溃），过肩镜头、慢推近景 | 崩溃瞬间，眼泪、发抖 |
| 全景/远景 | 冷战收尾 | 两人离得远，画面空旷、疏离 |

### 2. 氛围感走路景别（雨天/情绪向通用）
**逻辑**：远 → 中 → 近 → 特 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 远景 | 人物从远处走来，撑着伞，小小的身影 | 交代场景氛围 |
| 中景 | 正面/侧面走路，伞遮住半张脸，脚步匀速 | 看清楚人物+动作+环境 |
| 近景 | 拍到胸口以上，看表情、眼神 | 情绪开始进来 |
| 特写 | 鞋子踩进水洼，溅起水花 | 穿插细节，增加味道 |
| 远景 | 慢慢走远，背影变小 | 收尾留白 |

### 3. 枪战反转分镜
**逻辑**：铺垫 → 行动 → 反转收尾

| 阶段 | 内容 |
|------|------|
| 铺垫 | BOSS手持加特林疯狂输出；主角躲掩体后快速换弹，神情紧张；好友在另一掩体后比手势，吸引火力 |
| 行动 | 主角点头；弯腰从侧面绕过去；出现在BOSS侧翼；举枪瞄准；扣动扳机；小兵应声倒地 |
| 反转 | 好友走来问"都解决了？"→话音刚落，主角表情僵住→身后传来扳机扣动声→镜头从主角脸缓缓后移→黑洞洞的枪口抵着后脑勺 |

### 4. ❌ 错误示范（反派倒地）
> 反派中剑，缓缓倒地 → 躺在地上闭眼 → 主角松口气微笑 → 转身离开，画面渐黑
>
> **问题**：平白无奇，没有张力，没有反转

## 五、核心总结

写好Seedance提示词的本质：
- 站在AI角度，把创意拆成清晰、具体、有逻辑的执行指令
- 不用华丽辞藻，不用堆关键词
- 逻辑顺了，AI就能1:1还原你的想法

不管是AI工具还是实拍，核心差距不是工具，而是「把创意落地成精准指令」的能力。
