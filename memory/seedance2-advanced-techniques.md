## 一、Seedance 2.0 万能调整公式

### 核心公式
> 保留XX（满意的内容） + 把XX改成XX（要改的内容）

### 高频案例

| 问题 | 调整方法 |
|------|---------|
| 运镜太快/卡顿 | 保留整体色调、画面内容和分镜节奏，把所有镜头的运镜改为匀速缓慢运动，镜头切换时加入0.5秒的淡入淡出转场 |
| 画面跑偏/主体不对 | 保留整体的运镜、光影和分镜，严格以@图片1产品主体为核心，删除背景中多余杂物，主体始终在画面中心 |
| 质感不够/色调不对 | 保留所有分镜、运镜和画面内容，把冷色调换成暖黄色柔光，画面增加轻微胶片颗粒质感 |

## 四、分镜景别万能套路（实拍/AI通用）

### 1. 吵架戏万能景别
**逻辑**：中 → 近 → 特 → 中 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 中景 | 开场冲突，交代两人关系、距离、环境 | 让观众知道谁在吵、在哪吵 |
| 近景 | 情绪升级，切到面部表情 | 放大语气、眼神、微表情 |
| 特写/大特写 | 激烈爆发：眼睛、嘴唇、颤抖的手、眼泪 | 把情绪怼到观众脸上 |
| 中景 | 崩溃/哭（情绪崩溃），过肩镜头、慢推近景 | 崩溃瞬间，眼泪、发抖 |
| 全景/远景 | 冷战收尾 | 两人离得远，画面空旷、疏离 |

### 2. 氛围感走路景别（雨天/情绪向通用）
**逻辑**：远 → 中 → 近 → 特 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 远景 | 人物从远处走来，撑着伞，小小的身影 | 交代场景氛围 |
| 中景 | 正面/侧面走路，伞遮住半张脸，脚步匀速 | 看清楚人物+动作+环境 |
| 近景 | 拍到胸口以上，看表情、眼神 | 情绪开始进来 |
| 特写 | 鞋子踩进水洼，溅起水花 | 穿插细节，增加味道 |
| 远景 | 慢慢走远，背影变小 | 收尾留白 |

### 3. 枪战反转分镜
**逻辑**：铺垫 → 行动 → 反转收尾

| 阶段 | 内容 |
|------|------|
| 铺垫 | BOSS手持加特林疯狂输出；主角躲掩体后快速换弹，神情紧张；好友在另一掩体后比手势，吸引火力 |
| 行动 | 主角点头；弯腰从侧面绕过去；出现在BOSS侧翼；举枪瞄准；扣动扳机；小兵应声倒地 |
| 反转 | 好友走来问"都解决了？"→话音刚落，主角表情僵住→身后传来扳机扣动声→镜头从主角脸缓缓后移→黑洞洞的枪口抵着后脑勺 |

### 4. ❌ 错误示范（反派倒地）
> 反派中剑，缓缓倒地 → 躺在地上闭眼 → 主角松口气微笑 → 转身离开，画面渐黑
>
> **问题**：平白无奇，没有张力，没有反转

## 五、核心总结

写好Seedance提示词的本质：
- 站在AI角度，把创意拆成清晰、具体、有逻辑的执行指令
- 不用华丽辞藻，不用堆关键词
- 逻辑顺了，AI就能1:1还原你的想法

不管是AI工具还是实拍，核心差距不是工具，而是「把创意落地成精准指令」的能力。

## 六、九宫格/十二宫格一键成片（尤道子首发·2026-03）

### 核心玩法
用 NanoBanana 或其他图片工具生成九宫格/十二宫格分镜图，上传即梦 Seedance 2.0，选择「全能参考」，一句提示词即可生成完整视频。

### 九宫格分镜提示词模板
```
参考 @图片1 的九宫格分镜图片，按从左到右、从上到下的顺序演绎成一段15秒的视频，保持画面风格一致，转场流畅自然。
```

### 进阶九宫格提示词（指定运镜）
```
参考 @图片1 的九宫格分镜，按从左到右、从上到下的顺序演绎成15秒视频，保持画面风格一致，运镜以缓慢推拉为主，转场平滑，整体节奏舒缓。
```

### 电影级九宫格提示词（汽车TVC风格）
```
请依据SC1-SC9分镜图，采用电影级TVC摄影风格，生成一段12秒的广告片。
要求：
1. 画面采用变形宽银幕镜头(Anamorphic lens)质感，带轻微镜头光晕(Lens flare)和电影颗粒感
2. 确保车辆主体一致，光影呈现冷暖对比（如青橙色调）
```

### 十二宫格核心要点（尤道子方法）
- 即梦支持一次性生成12个镜头的九宫格，**无需分12次**
- 提示词格式：`根据图片的九宫格分镜，还原符合当下的剧情和人物神态和音效`
- 配合人物参考图可实现角色一致性
- 整体环境一致性满足，部分细节有丢失（建议选8-10秒而非15秒避免信息丢失）

### 尤道子九宫格示例（古风武侠）
```
根据图片九宫格分镜，生成一个完整的故事，音画同步。
镜头1：缓慢推镜，铁骑洪流向前涌动，马蹄扬起雪尘，旌旗猎猎飘动
镜头2：跟拍镜头，南衣女主回头张望，身后家丁追近，雪花在镜头前飞舞
镜头3：固定镜头，男的家丁抢南衣女主的玉镯子
...（依次每个格子一个镜头描述）
```

## 七、S-A-C-S-C五步提问法（尤道子首发·2026-03）

### 五步框架
| 步骤 | 要素 | 说明 |
|------|------|------|
| S | Subject 锚定主体 | 提供人物/产品/背景参考，@引用参考图，拒绝模糊描述 |
| A | Action 定义动作 | 清晰描述行为，聚焦物理状态变化，关键动作用英文动词 |
| C | Camera 规范运镜 | 专业摄影术语，明确景别+运镜方式，不用主观感受词 |
| S | Style 设定风格 | 统一光线+画质+艺术风格三项，用具体参数替代"高级感" |
| C | Constraints 添加约束 | 提前规避AI常犯错误（变形/穿帮/卡顿） |

### 约束词包（必加）
```
画面模糊、卡顿、人物变形、手部畸形、色彩溢出、穿帮镜头、逻辑混乱、画面抖动
```

## 八、Seedance 2.0 参数调优（2026主流配置）

### 基础参数设置
| 参数 | 建议值 | 说明 |
|------|--------|------|
| 视频时长 | 5-10秒 | 新手不超过8秒，容易崩 |
| 画幅比例 | 16:9(电影)/9:16(短视频)/1:1(方图) | 按平台选 |
| 帧率 | 24fps(电影)/30fps(流畅)/60fps(慢动作) | 按风格选 |
| Seed种子 | 固定数值 | 固定可复现，留空=创意 |
| 创意度 | 0.3-0.6 | 太高容易跑偏 |
| 运动强度 | 中等 | 高=快节奏打斗，低=慢镜头 |
| 参考强度 | 0.6-0.85 | 参考图/视频的权重 |

### @多模态参考标签语法
```
@Image1 - 参考图片1的外观/服装
@Image2 - 参考图片2的背景/道具
@Video1 - 延续视频1的动作
@Audio1 - 口型同步音频1
```
- 最多9张图 + 3段视频(≤15秒) + 3段音频(≤15秒)
- 编号按上传顺序从1开始

### 提示词加@参考的标准写法
```
参考@Image1的九宫格分镜，保持画面风格一致
参考@Image2的人物外貌和服装特征
延续@Video1的动作和运镜方式
```

## 九、动作词库：稳定vs不稳定

### 稳定动作词（优先用）
```
缓慢、连贯、自然、慢动作、平稳跟拍、脚步轻移、平稳横移、
微微低头、轻微拉远、轻微环绕、轻轻抬手、缓慢转身
```

### 不稳定动作词（尽量避免）
```
夸张、高速、复杂多人互动、大幅度扭曲、剧烈、快速横摇
```

### 动作描述原则
- 写「过程」而非「结果」
- ❌ `跳舞` → ✅ `随着节拍缓慢转身，手臂缓缓抬起划出弧线`
- ❌ `微笑` → ✅ `先是怔住，瞳孔微颤，而后笑意从嘴角漾开，染亮整双眼眸`
