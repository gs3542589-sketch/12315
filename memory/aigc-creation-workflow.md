## 一、完整创作总流程（7大阶段，42个细化步骤）

### 阶段1：需求锚定与立项
1. 明确创作目标：用途（引流/转化/宣传）+ 核心KPI
2. 锁定受众画像：年龄/性别/地域/痛点
3. 确定内容形式：文本/图像/音视频
4. 制定风格规范：视觉 + 语气
5. 划定边界约束：尺寸/时长/字数/禁忌
6. 竞品对标分析：拆解3-5个同赛道爆款

### 阶段2：前置素材与知识库准备
7. 建立参考库：图片/文本/音视频参考
8. 整理核心信息：参数/关键词库
9. 版权素材准备：无版权图库/音乐/字体
10. 构建专属知识库（RAG）：结构化、行业资料

### 阶段3：模型选型 + 专业提示词工程
11. 按场景选模型：文案/图像/视频/数字人/语音
12. 撰写结构化提示词：角色+任务+受众+结构+风格+约束
13. 增加约束与格式要求
14. 加入示例（Few-shot）
15. 设定输出格式

### 阶段4：AI批量生成 + 首轮粗筛
16. 批量生成初稿：同一提示词5-10版
17. 首轮粗筛：淘汰错误/跑题/差画质
18. 初步对齐需求：核对结构/卖点/风格

### 阶段5：人工精修 + 逻辑校准
19. 文本类精修：修正错误/优化流畅/加金句/排版
20. 图像类精修：修瑕疵/统一色调/加logo
21. 视频类精修：剪辑/配音/字幕/调色
22. 整体风格统一校验
23. 增加人性化细节
24. 最终整合定稿

### 阶段6：合规风控 + 版权审核
25. 内容安全审核：敏感/低俗/违规
26. 版权风险排查：肖像/音乐/字体/图片
27. 商标与品牌合规
28. 事实核查
29. 二次查重

### 阶段7：发布分发 + 数据复盘 + 迭代优化
30. 平台适配优化：抖音/小红书/公众号/电商
31. 发布配置：标题/话题/标签/封面
32. 数据埋点与监测
33. 复盘分析：高完播/低流量原因
34. 提炼爆款公式
35. 素材沉淀：提示词库/结构库/风格库
36. 下一轮迭代优化

## 二、分场景专项流程

### 流程A：AI文案创作
1. 定主题→定受众→定平台
2. 扒3个爆款标题结构
3. AI生成10组标题→筛3个
4. 生成正文框架：痛点→引入→价值→证明→行动指令
5. AI填内容→人工改逻辑→加金句→排版
6. 违禁词检查→查重→定稿

### 流程B：AI绘画/海报设计
1. 用途→尺寸→风格→配色
2. 找3张参考图定风格
3. 写提示词：主体+场景+光影+构图+画质+负面词
4. 批量出图→筛3张
5. PS精修：修手/脸/细节、加文字、排版
6. 商用合规检查→定稿

### 流程C：AI短视频全流程（核心）
1. 选题→定时长→定口播风格
2. AI写脚本：开头3秒→痛点→内容→结尾引导
3. 提取分镜：近景/中景/特写/转场
4. AI生成画面/数字人/混剪素材
5. 剪映合成：剪辑+配音+字幕+BGM
6. 调色+特效+封面→审核→发布

### 流程D：电商AIGC详情页
1. 产品核心卖点→用户痛点
2. 结构：首图→痛点→卖点→细节→参数→实拍→售后
3. AI生成文案+AI生成场景图
4. 套模板→排版→统一风格
5. 广告法审核→功效核实→上线

## 五、AI视频创作专项流程（根据用户要求定制）

### 标准流程：选题→脚本→分镜→参考图→视频

1. **选题定位**
   - 主题/情绪/目标人群/爆点

2. **脚本撰写**
   - 开头3秒/痛点/内容/结尾CTA
   - 输出：分镜表格（时间/画面/镜头/音效）

3. **参考图生成（NanoBanana）**
   - 人物图（保证角色一致性）
   - 场景图（保证背景一致性）
   - 道具图（关键道具特写）

4. **视频提示词（Seedance 2.0）**
   - @引用参考图
   - 动作时间轴拆分
   - 光影/镜头/风格
   - 约束词

5. **视频生成**
   - Seedance/即梦
   - 多段拼接

6. **精修发布**
   - 剪映合成：剪辑+字幕+BGM
   - 平台适配：抖音/小红书/B站
