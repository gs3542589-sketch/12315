## 一、LLM提示词基础结构

```
[Role/Context] + [Task] + [Constraints] + [Output Format]
```

### 角色设定
```python
"你是一位资深XX专家，拥有X年经验"
```

### 任务清晰
- ❌ 模糊："帮我写点什么"
- ✅ 具体："帮我写一个30秒的抖音带货脚本，产品是XX，目标人群是XX"

### 约束条件
- 字数限制、风格要求、禁止项

### 输出格式
- JSON、Markdown、表格、分点

## 二、Chain-of-Thought（思维链）

让AI逐步思考：
```
"这个问题请一步步思考，最后给出答案"
```

**适用场景**：复杂问题、推理、多步骤任务

---

## 三、Few-Shot（少样本学习）

给1-3个示例，让AI模仿：

```
示例1：
输入：XXX
输出：YYY

示例2：
输入：AAA
输出：BBB

现在请转换：
输入：CCC
```

## 四、图像生成提示词结构

```
[Subject主体] + [Style风格] + [Composition构图] + [Lighting光线] + [Technical技术参数]
```

### 主体描述
- ❌ "一只猫"
- ✅ "一只橘色英国短毛猫，绿色眼睛，脖子上有白色斑块"

### 风格关键词
```
照片级写实、8K、电影感、柔和自然光、浅景深
```

### 构图控制
```
三分法、对称构图、引导线构图
```

### 负面提示词（必加）
```
blurry, distorted, extra limbs, watermark, text, low quality
```

## 六、短视频脚本结构（核心！）

### 15-30秒结构
```
Hook(3s) → Content(20s) → CTA(2s)
```

### 30-60秒结构
```
Hook(3-5s) → Problem(5-10s) → Solution(15-30s) → Proof(5-10s) → CTA(3-5s)
```

### 黄金Hook公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 反常识 | "你以为XX，其实YY" | "你以为努力就能成功？方向比努力更重要" |
| 痛点 | "XX还YY？" | "忙了一整天，还是没钱？" |
| 数字承诺 | "X个方法解决YY" | "3个方法每天多2小时" |
| 反问 | "为什么XX？" | "为什么学了很多还是不行？" |
| 反常识 | "XX是最大的陷阱" | "做自己是最大的职业陷阱" |

### 结尾CTA
- "关注我，每天分享XX技巧"
- "点赞收藏，别错过"
- "评论区告诉我你的XX"

## 八、用户心理学（涨粉关键）

| 行为 | 内容类型 | 核心动机 |
|------|----------|----------|
| 点赞 | 共鸣型 | 情感触动，瞬时认同 |
| 收藏 | 工具型 | 未来价值，结构化可提取 |
| 评论 | 争议/情感型 | 情感表达，参与感 |
| 分享 | 利益/社交型 | 帮助他人，社交货币 |

### 涨粉内容特征
1. **结构清晰**：框架、逻辑、层次
2. **易于提取**：列表、步骤、关键术语
3. **多场景适用**：可迁移到多种场景

## 十一、常见错误

| 错误 | 问题 | 解决 |
|------|------|------|
| 太模糊 | 不可预测输出 | 添加具体性 |
| 太长 | 模型失去焦点 | 优先关键信息 |
| 矛盾 | 困惑模型 | 移除矛盾 |
| 无格式 | 不一致输出 | 指定格式 |
| 无示例 | 期望不清晰 | 添加few-shot |
