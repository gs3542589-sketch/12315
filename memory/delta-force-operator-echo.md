## 🔊 回响 (Echo) - 卢克・埃弗利 Luke Everly

### 基础信息
| 属性 | 详情 |
|------|------|
| 本名 | 卢克・埃弗利（Luke Everly） |
| 代号 | 回响（Echo） |
| 定位 | 侦察位（信息 / 控场） |
| 背景 | 前水下特种部队声呐专家，精通声纹识别，从深海战场转战陆地 |

### 核心技能

| 技能类型 | 名称 | 效果 |
|----------|------|------|
| **被动** | 声音感知 | 扩大声音收集范围，更早听到脚步声/换弹声 |
| **战术道具** | 次声波干扰器 | 降低敌人拾音范围，造成软控 |
| **大招** | 回声探测器 | 周期性扫描区域，标记发出声音的敌人（无视简单掩体） |

### 定位分析
- **角色定位**: 侦察位，信息收集
- **核心价值**: 声音感知+区域扫描，适合收集情报、提前预警
- **克制**: 依赖声音定位的敌人
- **被克制**: 静音移动的敌人

### 内容创作角度
- 攻略价值：高（侦察技巧、声音感知利用）
- 玩法亮点：声波探测机制独特
- 视觉效果：声呐/雷达风格特效
