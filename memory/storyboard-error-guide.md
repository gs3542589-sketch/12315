# 分镜制作避坑指南：4个高频错误+解决方案

## 来源

## 一、错误1：镜头语言不清晰（景别滥用）

### 错误表现
全程只用中景，缺乏特写、全景变化，画面单调、情绪没层次，观众抓不住重点。

### 解决方案：景别变化万能公式（对话场景）

> 全景(定位置) → 近景A(台词) → 近景B(反应) → 特写(关键台词)

| 景别 | 内容 | 作用 |
|------|------|------|
| 全景 | 交代两人的位置、环境 | 建立空间认知 |
| 近景A | 拍说话人的表情、语气 | 传递台词情绪 |
| 近景B | 拍听话人的反应 | 让对话有来有回 |
| 特写 | 关键台词、情绪爆发时 | 把情绪怼到观众脸上 |

---

## 二、错误2：轴线混乱（越轴导致空间错乱）

### 错误表现
镜头越过了两人之间的180度轴线，导致人物左右位置、视线方向完全颠倒，观众产生空间错乱感。

### 解决方案：严格遵循180度轴线规则

- 在两个角色之间画一条虚拟「轴线」
- 所有机位必须在轴线**同一侧**拍摄
- 角色的空间位置、视线方向始终一致
- 必须越轴时：用「运动镜头、中性镜头、人物转头」做过渡

---

## 三、错误3：画面信息过载/不足

### 错误表现
单镜头塞入过多元素→焦点模糊；或信息不足→交代不清场景和人物。

### 解决方案：用构图+光影引导注意力

| 方法 | 操作 |
|------|------|
| 做减法 | 删除多余元素，只保留核心主体 |
| 用光影 | 明暗对比突出重点（如给窗户打亮黄色光） |
| 用构图 | 三分法、引导线等，把视觉重心放核心 |
| 控信息 | 每个镜头只讲一件事 |

---

## 四、错误4：忽略剪辑连贯性（动作跳切）

### 错误表现
镜头之间缺乏匹配动作，出现动作跳切；忽略视线引导、运动方向连续性，导致画面割裂。

### 解决方案：保证运动/动作连贯性

| 方法 | 操作 |
|------|------|
| 动作匹配 | 上一个镜头的动作结尾 = 下一个镜头的动作开头 |
| 方向一致 | 人物/车辆运动方向在前后镜头必须统一 |
| 视线引导 | 人物视线方向连贯，观众视线跟着走 |
| 转场过渡 | 用匹配剪辑、运动转场让切换丝滑 |

---

## 核心总结

| 避坑点 | 核心原则 |
|--------|---------|
| 景别 | 用变化传递情绪，避免单调 |
| 轴线 | 用规则保证空间，避免错乱 |
| 信息 | 用主次聚焦重点，避免混乱 |
| 剪辑 | 用连贯强化代入，避免跳切 |
