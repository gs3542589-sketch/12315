## 一、景别套路

### 吵架戏万能景别
**逻辑**：中 → 近 → 特 → 中 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 中景 | 开场冲突，交代两人关系、距离、环境 | 让观众知道谁在吵、在哪吵 |
| 近景 | 情绪升级，切到面部表情 | 放大语气、眼神、微表情 |
| 特写/大特写 | 激烈爆发：眼睛、嘴唇、颤抖的手、眼泪 | 把情绪怼到观众脸上 |
| 中景 | 崩溃/哭（情绪崩溃），过肩镜头、慢推近景 | 崩溃瞬间，眼泪、发抖 |
| 全景/远景 | 冷战收尾 | 两人离得远，画面空旷、疏离 |

### 氛围感走路景别（雨天/情绪向）
**逻辑**：远 → 中 → 近 → 特 → 远

| 景别 | 内容 | 作用 |
|------|------|------|
| 远景 | 人物从远处走来，撑着伞，小小的身影 | 交代场景氛围 |
| 中景 | 正面/侧面走路，伞遮住半张脸，脚步匀速 | 看清楚人物+动作+环境 |
| 近景 | 拍到胸口以上，看表情、眼神 | 情绪开始进来 |
| 特写 | 鞋子踩进水洼，溅起水花 | 穿插细节，增加味道 |
| 远景 | 慢慢走远，背影变小 | 收尾留白 |

### 枪战反转分镜
**逻辑**：铺垫 → 行动 → 反转收尾

| 阶段 | 内容 |
|------|------|
| 铺垫 | BOSS手持加特林疯狂输出；主角躲掩体后快速换弹，神情紧张；好友在另一掩体后比手势，吸引火力 |
| 行动 | 主角点头；弯腰从侧面绕过去；出现在BOSS侧翼；举枪瞄准；扣动扳机；小兵应声倒地 |
| 反转 | 好友走来问"都解决了？"→话音刚落，主角表情僵住→身后传来扳机扣动声→镜头从主角脸缓缓后移→黑洞洞的枪口抵着后脑勺 |

### 正反打镜头（180度轴线规则）

#### 基础正反打（纯对话，聚焦情绪）
| 镜头 | 视角 | 画面 | 作用 |
|------|------|------|------|
| 正拍 | 从A的视角拍B | B的脸/上半身 | 让观众看到B在说话 |
| 反打 | 从B的视角拍A | A的脸/上半身 | 让观众看到A的反应 |

#### 过肩正反打（最常用，锚定空间）
| 镜头 | 视角 | 画面 | 作用 |
|------|------|------|------|
| 过肩正拍 | 越过A的肩膀拍B | B的脸清晰，A的肩膀/后脑勺在画面一侧 | 锚定两人空间位置 |
| 过肩反打 | 越过B的肩膀拍A | A的脸清晰，B的肩膀/后脑勺在画面一侧 | 强化空间连贯性 |

**核心原则**：所有机位必须在两人连线的同一侧，**绝对不能越轴**

## 二、九宫格/十二宫格成片

### 核心玩法
用图片工具生成九宫格分镜图 → 上传即梦 Seedance 2.0 → 选择「全能参考」→ 一句提示词生成完整视频

### 九宫格提示词模板（基础版）
```
参考 @图片1 的九宫格分镜图片，按从左到右、从上到下的顺序演绎成一段15秒的视频，保持画面风格一致，转场流畅自然。
```

### 九宫格提示词模板（指定运镜）
```
参考 @图片1 的九宫格分镜，按从左到右、从上到下的顺序演绎成15秒视频，保持画面风格一致，运镜以缓慢推拉为主，转场平滑，整体节奏舒缓。
```

### 九宫格提示词模板（电影TVC风格）
```
请依据SC1-SC9分镜图，采用电影级TVC摄影风格，生成一段12秒的广告片。
要求：
1. 画面采用变形宽银幕镜头(Anamorphic lens)质感，带轻微镜头光晕(Lens flare)和电影颗粒感
2. 确保车辆主体一致，光影呈现冷暖对比（如青橙色调）
```

### 尤道子九宫格示例（古风武侠）
```
根据图片九宫格分镜，生成一个完整的故事，音画同步。
镜头1：缓慢推镜，铁骑洪流向前涌动，马蹄扬起雪尘，旌旗猎猎飘动
镜头2：跟拍镜头，南衣女主回头张望，身后家丁追近，雪花在镜头前飞舞
镜头3：固定镜头，男的家丁抢南衣女主的玉镯子
...（依次每个格子一个镜头描述）
```

### 十二宫格核心要点
- 即梦支持一次性生成12个镜头，**无需分12次**
- 提示词：`根据图片的九宫格分镜，还原符合当下的剧情和人物神态和音效`
- 配合人物参考图可实现角色一致性
- **建议选8-10秒而非15秒**（避免信息丢失）

## 三、完整工作流（小说/故事→视频）

### Step 1：分镜拆解（DeepSeek/豆包）
- 单分镜固定时长：**15秒**（即梦最优生成时长）
- 每个分镜包含4要素：核心情节动作 + 出场人物 + 场景环境 + 关键道具
- 动作描写：起始→过程→结尾，不写笼统模糊内容
- 分镜衔接：前一个分镜结尾动作 = 后一个分镜起始动作

### Step 2：定版素材制作（文生图）
**防崩脸的核心！跳过这步出问题概率90%**

| 类型 | 内容 |
|------|------|
| 人物三视图 | 正面+侧面+背面，全部统一五官/发型/服饰/配饰/体态 |
| 关键道具 | 佩剑、玉佩、核心物品 |
| 核心场景 | 古宅、山林、朝堂大殿等 |

生成后**反复调整直到完全满意**，后续所有视频生成都以定版图为基准。

### Step 3：逐镜生成（Seedance 2.0 图生视频）
- 以对应的定版图作为起始参考
- 逐镜生成 15 秒，**逐帧检查**：崩脸/服饰跑偏/动作不符/场景不一致
- **一致性锁链**：前一个分镜的定稿视频 = 后一个分镜的一致性参考
- 不满意就**调整提示词重新生成**，不要批量生产后返工

### Step 4：剪映拼接
- 按叙事顺序导入所有15秒片段
- 调整播放速度、转场效果
- 配 BGM + 人声配音 + 音效
- 统一色调，导出成片

## 五、万能分镜模板

### 好物/产品赛道
| 时段 | 内容 | 镜头操作 |
|------|------|---------|
| 0-3秒 | 黄金开场：核心卖点特写 | 快速推近，直击痛点 |
| 3-8秒 | 细节展示：360度环绕 | 匀速环绕，侧逆光 |
| 8-12秒 | 场景植入 | 缓慢拉远，放入真实场景 |
| 12-15秒 | 收尾引导 | 固定镜头，搭配slogan |

### 氛围/情绪赛道
| 时段 | 内容 | 镜头操作 |
|------|------|---------|
| 0-3秒 | 开场定调：大场景空镜 | 缓慢运镜，定好色调氛围 |
| 3-7秒 | 细节特写 | 微距拍环境细节 |
| 7-12秒 | 主体植入 | 镜头拉远，带出主体 |
| 12-15秒 | 收尾留白 | 固定长镜头，慢慢暗下 |
