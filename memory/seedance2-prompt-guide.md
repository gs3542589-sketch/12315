## 一、底层逻辑：Seedance 2.0 吃什么提示词？

**核心认知**：Seedance 2.0 不是关键词堆砌工具，而是"听话的执行导演"。

**本质区别**：
- 不吃零散的关键词堆砌
- 只吃完整的、有时间线、有画面逻辑、有细节指令的导演式叙事文本

**正确姿势**：把自己当导演，像对着摄影师说话一样，清清楚楚告诉它：每一秒要拍什么、怎么拍、用什么光、什么节奏、什么感觉。

## 二、把抽象感觉拆解成具体画面

### 新手高频抽象词「翻译表」

| 抽象词 | 具体拆解 |
|--------|---------|
| **高级感** | 低饱和度莫兰迪色系 + 45度侧逆光勾勒主体轮廓 + 浅景深虚化背景 + 画面留白占比60% + 匀速慢镜头运镜 |
| **氛围感** | 暖黄色柔光 + 轻微胶片颗粒质感 + 前景虚化遮挡 + 慢门动态模糊 + 环境音同步 |
| **电影感** | 2.35:1宽画幅 + 明暗对比强烈的伦勃朗光 + 镜头缓慢推拉 + 低饱和影调 + 前景纵深构图 |
| **治愈感** | 明亮柔和的自然光 + 高通透浅色系画面 + 微距特写细节 + 缓慢平移运镜 + 白噪音背景音 |

### 视觉元素拆解四步法

1. **定核心主体**：明确画面里的主角是什么，要占多少画面比例
2. **定光影质感**：明确用什么光、什么色调、什么画面质感
3. **定运镜节奏**：明确镜头怎么动、动多快、什么时候切换
4. **定环境氛围**：明确背景是什么、有什么辅助元素、整体情绪是什么

### 举例（治愈感果茶）

❌ 错误：`治愈感果茶`（空话）

✅ 正确：
> 一杯装在透明玻璃杯里的蜜桃果茶，占画面中心70%的比例；窗边明亮柔和的自然光打在杯子上，茶汤通透有光泽，整体是浅粉色暖色调；镜头缓慢匀速向杯口推近，全程无卡顿；背景是虚化的白色窗帘，画面干净通透，整体氛围治愈安静

## 三、掐着15秒写分镜

**核心原则**：15秒视频，分镜控制在 **3-5段**，每段至少2-3秒，少而精。

**黄金3秒定律**：开头3秒必须抓住用户眼球。

### 15秒万能分镜模板

#### 好物展示赛道（美妆、食品、家居）

| 时段 | 内容 | 镜头操作 |
|------|------|---------|
| 0-3秒 | 黄金开场：产品核心卖点特写 | 镜头快速推近，直击痛点（如爆汁果肉慢动作） |
| 3-8秒 | 细节展示：360度环绕 | 匀速环绕运镜，侧逆光打亮产品 |
| 8-12秒 | 场景植入 | 镜头缓慢拉远，产品放入真实使用场景 |
| 12-15秒 | 收尾引导 | 固定镜头，产品主体清晰，搭配极简slogan |

#### 氛围感短片赛道（风景、治愈、情绪向）

| 时段 | 内容 | 镜头操作 |
|------|------|---------|
| 0-3秒 | 开场定调：大场景空镜 | 缓慢运镜，定好整体色调和氛围 |
| 3-7秒 | 细节特写 | 微距拍环境细节（飘落花瓣、晃动光影） |
| 7-12秒 | 主体植入 | 镜头拉远，带出画面主体（窗边的人、桌上的书） |
| 12-15秒 | 收尾留白 | 固定长镜头，画面慢慢暗下，留足情绪空间 |

## 四、用自然语言写，像跟朋友说话一样

### ✅ 正确写法（有时间线、有逻辑）
> 暖色调拍摄蜜桃果茶展示视频，0-3秒镜头快速推近到杯口，拍果肉爆汁的慢动作特写；3-8秒镜头绕着杯子匀速缓慢旋转，侧面柔光打亮杯身，突出玻璃质感和茶汤的通透度；8-12秒镜头慢慢拉远，杯子放在原木桌面上，背景是虚化的白色窗帘；12-15秒固定镜头，画面干净安静，整体氛围治愈温暖。

### ❌ 错误写法（堆砌关键词、无逻辑）
> 果茶、高端、旋转、特写、暖光、木桌、治愈、蒸汽。

## 五、参考素材的正确用法

### 核心原则
不只是贴素材，更要明确标注素材的用途，告诉AI要参考素材的什么内容，复刻到什么程度。

### ✅ 正确素材用法
> @图片1 作为蜜桃果茶的主体参考，严格保留杯子的造型、logo和果肉的颜色；@视频1 的360度匀速环绕运镜完全复刻，运镜速度保持一致；@音频1 的轻音乐作为背景音，镜头切换卡点音乐的鼓点，每4秒切换一次镜头。

### ❌ 错误素材用法
> @图片1@视频1@音频1 果茶展示。
